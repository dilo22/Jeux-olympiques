# Application de Gestion des Jeux Olympiques 2024

Cette application permet de gérer et de visualiser les données des Jeux Olympiques de Paris 2024, en fournissant des informations détaillées sur les délégations, les athlètes, les disciplines et les résultats des compétitions. 

# Prérequis

-Python 3 
-Bibliothèques nécessaires : Flask, Pillow, Tkinter, Psycopg2



# Structure des dossiers

- `main.py` : Le fichier principal à exécuter pour lancer l'application.
- `images/` : Contient toutes les images utilisées dans l'application, y compris les drapeaux et les icônes.
- `aide/` : Contient les fichiers de l'application intermédiaire, servant de support ou de documentation à l'utilisateur.
- `insertion/` : Contient toutes les données nécessaires pour l’application.

# Lancer l'Application

Pour démarrer l'application, exécutez le fichier `main.py` :

python main.py

Cela ouvrira l'interface principale de l'application, où vous pourrez naviguer entre les différentes fonctionnalités, afficher les délégations, consulter les résultats des athlètes et visualiser le tableau des médailles.
